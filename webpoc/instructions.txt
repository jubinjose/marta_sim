Note: Those not on Windows, please use appropriate classpath separator for their OS. Believe it is : for Unix and MacOS

To Compile
-----------
javac -cp spark-core-2.8.0-jar-with-dependencies.jar *.java

To Run (please don't close the window after you run)
-----------------------------------------------------
java -cp spark-core-2.8.0-jar-with-dependencies.jar;. Main Jubin

To Test
---------
Launch browser and try below websites

a) http://localhost:4567
   
   This should echo back Jubin -> Shows how command line arguments retained and used by the web app

b) http://localhost:4567/concat/Jose

   Will keep appending to previous string and return value ->   shows how state can be maintained
   You can keep hitting the URL with different strings at end
